.kmp file format

全ての空白は無視されます．
行頭にハイフン'-'を使用することにより，その行は全て無視されます．
大文字と小文字は区別されます．
ファイルのエンコードはshift JISにしてください．
IDは全てqwerty配列のものが適用されます．



#!iddef : A : B
説明:ID"B"の名をAにする．
同一IDで複数#!iddefした場合，最後のものが適用されます．

#dirdef : {ID} : {R or L}
説明：キーを打つ手を明示します．
このキーでは，Cmd==1stSendの時，キーが2ndSendのRLによって変化します．


ID
ID説明
システム既定のIDを使用するには，IDの初めに ? を使用します．
?から初まる全てのIDは予約されているので使用してはいけません．
#!iddefで既定したIDを使用するには，IDの初めに ! を使用します．
同一名称のIDは使用できません．(?XXXと!XXXはOK)
ID名が一文字のみならば，?及び!は不要です．ただし，記号は使えません．

{ID1} : {Cmd} : {ID2}
IDは & を使用することにより，複数列挙できます．
ID1 と ID2 が一致している場合，無視されます.
ID1が被った場合，最後のものが適用されます．

例
 
?A & ?V : Send : ?C & ?COMMA & ?Z
AかつVが同時に押された時，Cと,とZに置き換え．

Send  ID1をID2に置き換えます.

{ID1} : {Cmd} : {ID2}
If you use &, you can list up more than two.
If ID1 and ID2 is same, this line will be ignored.
If ID1 is already used, the latest one is used.

Example
 
?A & ?V : Send : ?C & ?COMMA & ?Z
When A and V are typed at the same time, replace A and V with C and , and Z.
