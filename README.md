# Q-ELF
Q-ELF配列はほとんどの音をキーボード中段のみで打つことが出来るキーボード配列です。SKKと親和性が高くなるよう設計されています。
キー配列が気に入らなければ再定義することも可能です。

Windows 11で動作確認をしています。
